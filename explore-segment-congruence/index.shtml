Explore Segment Congruence

If two segments have equal lengths, can you always transform the first segment so that its image lies exactly on top of the second? In this activity you’ll experiment by using different transformations

1 Superpose `overline(AB)` on `overline(CD)`

Using this sketch, you’ll demonstrate that pairs of segments are congruent by finding multiple ways to combine transformations to place one segment exactly on top of the other.

 

2 Reflection Only

Using this sketch, you’ll construct reflections to demonstrate that pairs of segments are congruent.
