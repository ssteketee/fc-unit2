Experiment with Segment Congruence

▿︎ Sketch

Using this sketch, you’ll demonstrate that pairs of segments are congruent by finding multiple ways to combine transformations to place one segment exactly on top of the other.

 

▿︎ Reflection Only

Using this sketch, you’ll construct reflections to demonstrate that pairs of segments are congruent.

 

▹︎︎ The Fine Print

Requirements:
