Mirror, Mirror

▿︎ Orientation

Each page of this activity has an independent variable that can move along a restricted domain (a triangle). You’ll transform the variable, using a different function on each page, and investigate whether the dependent variable has the same orientation (both clockwise or both counter-clockwise) or a different orientation compared to the independent variable.

 

▹︎ Explore Multiple Reflections

On the pages of this activity you’ll reflect the independent variable (along with its domain) once, twice, or three times. In each case you’ll see if you can find a function from a different transformation family that produces the exact same result as the combination of reflect functions you used.

 

▹︎ Construct Multiple Reflections

On each page of this activity you’ll try to construct multiple reflect functions to produce the exact same result as other transformations. The functions you'll try to construct this way include a rotate function (page 2), a translate function (page 3), a glide reflection (page 4), and a dilate function (page 5). (Hint: all of these except one are possible.)
